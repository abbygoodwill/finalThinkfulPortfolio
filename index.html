What I have been working on

Hot or Cold App

This app is a game designed by Thinkful. It is a game that selects a number. Then the user has to guess the value of this number. As the user continues to guess, the app returns the words, "hot" or "cold" based on how close the guess number is to the selected number.

Product API Finder

This project works alongside the Etsy API. It allows the user to enter in a type of product that they are looking for. Then it returns the top products with that title listed on the Etsy website.

Quiz App

This project was based off of a quiz I had in biology. It is a basic quiz app with 6 questions and as the user continues to do the quiz, the app will calculate the grade the user currently has based off of how many questions they have completed and gotten correct.
